Ngx datatable is a third party library we are using in our applicatios.

Installing

You can grab the latest release from the Releases Page in Github or via NPM.

npm install @swimlane/ngx-datatable

CSS

Import scss file to apply our theme style

@import '~@be-retail/be-theme/lib/styles/be-ngx-datatable-theme';

Apply the mixin

@include be-ngx-data-table($demo-theme);
